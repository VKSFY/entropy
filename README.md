# ENTROPY
 
A terminal-based diagnostic shell.
 
```
pip install -r requirements.txt
python main.py
```
 
Type `help` to begin.
 
---
 
State persists between sessions in `~/.entropy_state.json`.
Delete that file to start over.
 
The shell will not stop you from leaving. It will notice.
 
---
 
*Python 3.11+ — dependencies: `textual`, `rich`*
